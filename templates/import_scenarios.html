シナリオをインポート

YAML形式のシナリオ設定ファイルをインポートします。

インポート設定

ダウンロードしたシナリオ設定ファイルを編集してアップロードしてください。
キャンセル

YAML形式の例

network-health-check:
  devices:
    - router-01
    - switch-01
  commands:
    - show version
    - show running-config
    - show interfaces
    - show ip route
  description: ネットワークヘルスチェック
  group: monitoring
  delay: 2
  timeout: 30
  save_config: false

interface-configuration:
  devices:
    - router-01
  commands:
    - configure terminal
    - interface GigabitEthernet0/0
    - ip address 192.168.1.1 255.255.255.0
    - no shutdown
    - exit
  description: インターフェース設定
  group: configuration
  delay: 3
  timeout: 60
  save_config: true
