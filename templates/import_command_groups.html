コマンドグループをインポート

YAML形式のコマンドグループ設定ファイルをインポートします。

インポート設定

ダウンロードしたコマンドグループ設定ファイルを編集してアップロードしてください。
キャンセル

YAML形式の例

basic-config:
  commands:
    - show version
    - show running-config
    - show interfaces
  description: 基本設定確認コマンド
  group: basic

interface-config:
  commands:
    - show ip interface brief
    - show interfaces status
    - show interfaces description
  description: インターフェース設定コマンド
  group: interface

security-config:
  commands:
    - show access-lists
    - show ip ssh
    - show crypto isakmp sa
  description: セキュリティ設定コマンド
  group: security
